\# 🎙️ Voice Recorder Application using Python



\## 📌 Project Description

This is a simple \*\*Voice Recorder Application\*\* developed using \*\*Python\*\*.  

It provides a graphical user interface (GUI) where users can:



\- Start audio recording

\- Stop recording

\- Save recorded audio clips as `.wav` files

\- View saved recordings in a list

\- Play selected recordings



This project is beginner-friendly and useful for understanding:

\- GUI development in Python

\- Audio recording and playback

\- File handling

\- Event-driven programming



\---



\## 🛠️ Technologies Used

\- \*\*Python\*\*

\- \*\*Tkinter\*\* (for GUI)

\- \*\*Sounddevice\*\* (for recording and playback)

\- \*\*NumPy\*\* (for audio data handling)

\- \*\*SciPy\*\* (for saving `.wav` files)



\---



\## ✨ Features

\- Simple and user-friendly interface

\- Record audio through microphone

\- Stop and save recording

\- Play previously recorded audio clips

\- Displays recording time

\- Automatically saves recordings as `.wav` files
